# 🌙 Eid Salami Quiz Platform (Advanced)

এটি একটি হাই-এন্ড কুইজ প্ল্যাটফর্ম যেখানে ইউজাররা তাদের বন্ধুদের জন্য ইন্টারেক্টিভ কুইজ তৈরি করতে পারে।

### 🚀 ফিচারসমূহ:
- **Ready-made Questions:** ১০টি প্রি-সেট প্রশ্ন যা ইউজারকে টাইপ করার ঝামেলা থেকে বাঁচায়।
- **Premium Dark UI:** গ্লাস-মর্ফিজম এবং নিওন অ্যাকসেন্ট ডিজাইন।
- **Firebase Integration:** রিয়েল-টাইম ডাটা সেভিং এবং ইউনিক আইডি জেনারেশন।
- **Mobile Optimized:** বড় ফন্ট এবং টাচ-ফ্রেন্ডলি বাটন।

### 🛠 টেকনোলজি:
- HTML5, CSS3 (Custom Properties & Flexbox)
- JavaScript (ES6)
- Firebase Realtime Database

### 📖 কিভাবে ব্যবহার করবেন:
১. `index.html` ফাইলটি ওপেন করুন।
২. ফায়ারবেস কনফিগারেশনে আপনার নিজের `apiKey` এবং তথ্যগুলো দিন।
৩. ভেরসেল বা গিটহাব পেজে হোস্ট করুন।
